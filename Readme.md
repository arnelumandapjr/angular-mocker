# Angular Mocker

## a command line tool that generates mocks for your angular pipes, services, directives and components.

```js
// create mock for whole application
mocker --app-dir  ./

// create mock for a file
mocker app.component.ts
```

Command Options:
- --force / -f
- --verbose
- --src-dir / --srcDir
- --app-dir / --appDir
- --skip-barrel / --skipBarrel
